Privacy Policy

Last Updated: March 2026

Information We Collect

The Bible Verse Bot stores limited data required for functionality. This includes:

  • Discord Server IDs
  • Channel IDs where daily verses are sent

This data is stored so the bot can deliver daily Bible verses to the correct Discord channels.

Information We Do NOT Collect

The bot does not collect or store:

  • Personal user data
  • Messages
  • Email addresses
  • Discord account details

How Data Is Used

The stored server and channel IDs are only used to send scheduled daily Bible verses.

Data Sharing

No stored data is shared, sold, or distributed to any third parties.

Data Removal

If the bot is removed from a Discord server, the stored server data may be deleted.

Changes to This Policy

This privacy policy may be updated at any time.

Contact

If you have questions about this policy, contact the bot developer through Discord.
